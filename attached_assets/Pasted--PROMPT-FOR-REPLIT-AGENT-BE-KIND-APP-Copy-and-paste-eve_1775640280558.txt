## 1. GENERAL ARCHITECTURE

The app has **8 macro-modules**:
1. Authentication & User Profile
2. Restaurant & Digital Menu
3. Table Reservation
4. Delivery & Takeaway Orders (with real-time tracking)
5. E-Commerce (Product Shop)
6. Events & Workshops
7. **Loyalty Card System with STAMPS + POINTS** (key feature)
8. Admin Panel

Recommended stack:
- **Frontend:** React 18+ with TypeScript, Tailwind CSS, React Router
- **Backend:** Node.js with Express (or Next.js API Routes)
- **Database:** PostgreSQL with Prisma ORM (or Supabase)
- **Auth:** JWT with refresh token (or Supabase Auth)
- **Payments:** Stripe (checkout integration)
- **Maps:** Leaflet or Google Maps API (for rider tracking)
- **Real-time:** WebSocket or Supabase Realtime (for order tracking)
- **Image storage:** Cloudinary or Supabase Storage

## 8. LOYALTY CARD SYSTEM WITH STAMPS + POINTS (KEY FEATURE)

This is the core engagement feature. It works as a **digital loyalty card** with TWO parallel systems: **STAMPS** (like a physical stamp card) and **POINTS** (accumulated currency). Users see a visual card in the app that gets stamped as they do things.

### 8A. STAMP CARDS

The app features **multiple digital stamp cards**, each tied to a specific activity. When the user completes an activity, they earn a **stamp** on the relevant card. When a card is full, they unlock a **reward**.

**Stamp Card Types:**

| Card Name | Activity That Earns a Stamp | Stamps to Complete | Reward When Full |
|-----------|----------------------------|-------------------|-----------------|
| Colazione Card | Have breakfast at Be Kind | 8 stamps | Free breakfast of your choice |
| Brunch Card | Have brunch at Be Kind | 6 stamps | Free brunch for 1 |
| Delivery Card | Complete a delivery order | 10 stamps | Free delivery order up to €20 |
| Workshop Card | Attend a workshop/event | 4 stamps | Free access to next workshop |
| Shopping Card | Make an e-commerce purchase | 5 stamps | 30% off next purchase |
| Review Card | Leave a review (any type) | 10 stamps | Dessert of your choice free |

**Visual Design of Stamp Cards:**
- Each card looks like a physical loyalty card with empty circles/slots
- When user earns a stamp, the circle fills in with a beautiful animation (stamp/seal effect)
- Each card has its own illustration/icon (coffee cup for breakfast, paintbrush for workshop, etc.)
- Cards are displayed horizontally swipeable in the loyalty section
- Show progress: "5/8 stamps" with visual filled/empty dots
- When card is completed: celebration animation, reward unlocked notification
- Completed cards go to "Completed Cards" archive, and a new empty card starts automatically

**How Stamps Are Credited:**
- For in-restaurant activities (breakfast, brunch, dine-in): staff scans user's QR code from the app, OR user shows order confirmation
- For delivery/takeaway: automatic after order is marked as delivered/picked up
- For workshops: automatic after QR code check-in at event entrance
- For e-commerce: automatic after shipping confirmation
- For reviews: automatic after review is submitted

### 8B. POINTS SYSTEM

In parallel to stamps, users also earn **Be Kind Points** for every action. Points are a global currency that can be redeemed for rewards from a catalog.

**Actions That Earn Points:**

| Action | Points Earned |
|--------|--------------|
| Complete a delivery order | 1 point per €1 spent |
| Complete a takeaway order | 1 point per €1 spent |
| Dine-in (breakfast, brunch, lunch, dinner) | 1 point per €1 spent |
| E-commerce purchase | 1 point per €1 spent |
| Book a table + show up (check-in) | 20 points |
| Leave a dish review | 10 points |
| Leave an order review | 15 points |
| Leave an event review | 15 points |
| Attend a workshop/event | 30 points |
| Refer a friend (referral) | 50 points (when friend completes first order) |
| Birthday bonus | 100 bonus points (automatically on birthday) |
| First order ever (welcome bonus) | 50 bonus points |
| Complete a stamp card | 50 bonus points |

All point values must be **configurable from the admin panel**.

### 8C. LOYALTY TIERS

As users accumulate points, they level up through tiers that unlock increasing benefits:

| Tier | Threshold | Benefits |
|------|-----------|----------|
| 🌱 Seed | 0 – 499 points | Standard point accumulation |
| 🌿 Sprout | 500 – 1,499 points | Points multiplier x1.2, early access to events, 5% permanent discount |
| 🌸 Bloom | 1,500 – 3,999 points | Points multiplier x1.5, free delivery, 10% discount, annual exclusive event |
| 🌳 Tree | 4,000+ points | Points multiplier x2, all Bloom benefits + exclusive gifts, priority reservations |

The tier names match the Be Kind nature/sustainability theme.

### 8D. REWARDS CATALOG (Points Redemption)

A dedicated in-app section called "Premia Te Stesso" (Reward Yourself) where users can redeem points:

- 100 points → Free coffee or tea
- 200 points → 10% off next order
- 300 points → Free dessert
- 500 points → Free shipping on e-commerce
- 800 points → Free access to a workshop
- 1,000 points → €15 voucher for the restaurant
- 1,500 points → Dinner for 2 on the house
- 3,000 points → Exclusive experience (dinner with the Chef)

Redemption happens during checkout (for discounts/vouchers) or from the "Premia Te Stesso" section in profile.

### 8E. LOYALTY SECTION UI

The loyalty section in the app ("La Mia Card" or "Be Kind Card") must contain:
- **Top area:** user's name, current tier with icon (🌱🌿🌸🌳), total points balance, progress bar to next tier
- **Stamp Cards area:** horizontal carousel of all active stamp cards with progress dots. Tap a card to see full detail.
- **QR Code:** the user's personal QR code that staff scan for in-restaurant stamp credits (always accessible with one tap)
- **Points History:** chronological log of all points earned and spent, with action detail and date
- **Rewards Catalog:** browsable list of available rewards with "Redeem" button
- **Completed Cards Archive:** view past completed stamp cards
